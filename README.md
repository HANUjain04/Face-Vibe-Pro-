FaceVibe Pro is a premium, frontend-first web application that simulates a highly advanced AI facial intelligence platform. Inspired by the sleekest cyber-aesthetic interfaces, it transforms a simple face scan into a comprehensive, interactive "LooksMax" and dermatology report.

Whether you are uploading a photo or using the live AR-style camera feed, FaceVibe Pro delivers a futuristic UI/UX complete with glassmorphism, neon accents, and smooth micro-interactions.

✨ Key Features

📸 AR-Style Live Scanner: Uses your device's camera to simulate a high-tech facial landmark mapping and scanning process (or allows file uploads).

📊 Intelligence Dashboard: Generates a comprehensive "LooksMax" score, visual age prediction, and an interactive SVG radar chart breaking down specific facial features (Jawline, Symmetry, Eyes, etc.).

⏳ Time Travel (Age Simulator): An interactive slider that seamlessly applies dynamic CSS filters and texture overlays to simulate how your face will age from 20 to 80 years old.

💧 Deep Skin Analysis: Maps out your face with simulated AR heatmaps, scoring hydration, elasticity, and smoothness.

🧴 Smart Routine Generator: Automatically prescribes a personalized AM/PM skincare regimen based on the results of your skin analysis.

🎭 Global Celebrity Match: Compares your facial structure against a mock database of celebrities to find your closest high-profile lookalikes.

📈 Evolution Tracker: Saves your past scans locally to track your glow-up journey over time via an interactive line chart.

💌 Shareable Glow-Up Cards: Generates and downloads an aesthetic, Instagram-ready "Vibe Card" of your results using built-in canvas rendering.

💻 Tech Stack

This project is built to be incredibly lightweight and fast, running entirely in the browser without the need for a complex backend.

Core: HTML5, Vanilla JavaScript (ES6+)

Styling: Tailwind CSS (via CDN) for rapid, responsive, and highly customized styling.

UI/UX: Custom CSS Glassmorphism (backdrop-filter), CSS Animations, and dynamic SVG generation for charts.

Icons: Lucide Icons for crisp, modern iconography.

Utilities: html2canvas for exporting the DOM into shareable image files.
